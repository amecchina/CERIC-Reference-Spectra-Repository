Reference Spectra Repository

The current repository provides centralized access to the reference spectra acquired at CERIC facilities, supporting data interpretability and reproducibility for researchers. See the right image for an example XDI spectrum plot.

X-ray absorption spectroscopy (XAS) data are currently available for the LISA@ESRF, XAS@ELETTRA, and ASTRA@SOLARIS beamlines. Data can also be retrieved via Python API, as shown in the example notebook.
